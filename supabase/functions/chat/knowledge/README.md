# Knowledge Base Directory

Place `.txt` or `.md` files here to embed them into the AI trading assistant's knowledge.

The edge function automatically reads all text files in this directory and includes them in the system prompt.

## How to use:
1. Convert your PDF to `.txt` or `.md` format
2. Place the file in this directory (e.g., `trading-guide.txt`)
3. The AI will automatically have access to the content on next deploy

## Supported formats:
- `.txt` - Plain text
- `.md` - Markdown
